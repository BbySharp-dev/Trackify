# Trackify

Trackify là một hệ thống theo dõi giá sản phẩm từ các trang thương mại điện tử tại Việt Nam.

## Công nghệ sử dụng
- Ngôn ngữ: C# (.NET 8)
- ORM: Entity Framework Core
- Database: SQL Server (Azure SQL Edge cho Mac M2)
- Scraping: HtmlAgilityPack, Playwright
- HTTP Resilience: Polly
- Logging: Serilog
- Testing: xUnit, Moq, WebApplicationFactory
- Auth: JWT Bearer Token
- Containerization: Docker, Docker Compose
